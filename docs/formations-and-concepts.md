## Concept Tags (Formation Matching)

컨셉은 태그로 포메이션과 매칭됩니다:

```typescript
// 예시: Mesh 컨셉
tags: ['2x2', 'trips', 'bunch', 'spread']

// 포메이션 → 태그 매핑
'spread'  → ['2x2', 'spread']
'trips'   → ['trips', '3x1']
'bunch'   → ['bunch', 'trips']
'pistol'  → ['pistol', '2x2']
```

### 주요 태그
| Tag | 설명 |
|-----|------|
| `2x2` | 양쪽 2명 리시버 |
| `3x1` | 한쪽 3명 리시버 |
| `trips` | Trips 포메이션 |
| `bunch` | Bunch 포메이션 |
| `spread` | Spread 계열 |
| `pro` | Pro Style |
| `heavy` | Heavy 세트 |
| `empty` | Empty 백필드 |
| `te_strong` | TE 강측 |
| `motion` | 모션 필요 |
