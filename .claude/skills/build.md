# /build - 빌드 확인

이 스킬은 프로젝트 빌드를 실행하고 결과를 보고합니다.

## 실행 단계

1. **빌드 실행**
   ```bash
   cd c:/FOC/editor && npm run build
   ```

2. **결과 보고**
   - 성공 시: "빌드 성공! 배포 준비 완료."
   - 실패 시: 에러 내용을 분석하여 보고
     - TypeScript 에러: 파일 경로와 에러 내용
     - ESLint 에러: 규칙 위반 내용
     - Import 에러: 누락된 모듈 또는 잘못된 경로

3. **에러 수정 제안** (실패 시)
   - 에러 유형에 따른 수정 방법 제안
   - 사용자가 원하면 자동 수정 시도

## 옵션

- `/build --fix`: 빌드 에러 발생 시 자동 수정 시도
- `/build --lint`: 빌드 전 린트 체크도 실행
